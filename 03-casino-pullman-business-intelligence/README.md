## 📋 Resumen del Proyecto

Análisis integral de **Business Intelligence** para el complejo **Casino Pullman City Center** en Rosario, Argentina. Este proyecto combina análisis de datos avanzado con storytelling para optimizar la explotación hotelera del mayor casino de América del Sur.

### 🎯 Objetivos
- Analizar performance del complejo gaming-hotelero
- Desarrollar storytelling con datos para stakeholders
- Optimizar revenue streams y customer segmentation
- Crear dashboards ejecutivos para toma de decisiones

---

## 🏢 Información del Complejo

**Casino Pullman City Center** - Rosario, Argentina
- 🎰 **Gaming**: Mayor casino de América del Sur
- 🏨 **Hotel**: Pullman 5 estrellas (200 habitaciones)
- 🎲 **Capacidad**: Sala de juego para 9,000 personas
- 🎰 **Máquinas**: 3,000 slot machines + mesas de juego
- 🍽️ **F&B**: Múltiples restaurantes y bares
- 🛍️ **Retail**: Centro comercial integrado
- 🎪 **Eventos**: Sala de convenciones y espectáculos
